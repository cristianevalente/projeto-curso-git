Galera, pra quem tem windows e ficou perdida nsa aulas, fiz esse post pra dar umas dicas de links e ferramentas. Se caso não conseguiu reproduzir as orientações do professor seguem:



- GIT no WINDOWS 
Ele tá usando o terminal do Linux. Para ter o mesmo resultado, vejam esse vídeo:

https://www.youtube.com/watch?v=SOxafinthys



Enquanto veem o vídeo, baixem o Git no Windows:

https://git-scm.com/downloads



- VISUAL STUDIO CODE
o VSC é a IDE mais usada hoje em dia (na minha opinião), então baixem e tenha o hábito de usa-lá, futuramente ela vai ser requisitada novamente:

https://code.visualstudio.com/download



- HTML
O professor usou um "texto padrão" de HTML, que serve pra orientar o navegador, ele vem por padrão do VSC, basta escrever hmtl:5 e dar enter no inicio do documento, e pronto!



Ps1.: O que ele deleta é "perfumaria".



Ps2.: Sugestão de curso free de HMTL/CSS: https://www.youtube.com/playlist?list=PLHz_AreHm4dlAnJ_jJtV29RFxnPHDuk9o
